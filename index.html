Designer's Scaling Units

intro

by use of css vars and the css calc functionality we can make a frankenstein, rube goldberg style accommodation for designers.

granted the original use of css vars was eciplsed by the use of the SASS, and even still all examples I have found on the web basically deal with colors and nothing else, which may have been a limiting factor when css vars came out. in any case enjoy.

CSS Absolute Lengths

  • cm - centimeters
  • mm - millimeters
  • in - inches (1in = 96px = 2.54cm)
  • px - pixels (1px = 1/96th of 1in)
  • pt - points (1pt = 1/72 of 1in)
  • pc - picas (1pc = 12 pt)

CSS Relative Lengths

  • em - Relative to the font-size of the element (2em means 2 times the size of the current font)
  • ex - Relative to the x-height of the current font (rarely used)
  • ch - Relative to the width of the "0" (zero)
  • rem - Relative to font-size of the root element
  • vw - Relative to 1% of the width of the viewport*
  • vh - Relative to 1% of the height of the viewport*
  • vmin - Relative to 1% of viewport's* smaller dimension
  • vmax - Relative to 1% of viewport's* larger dimension
  • % - Relative to the parent element
w3schools reference

a tad

make the p tag font-size a tad larger

This is a tad larger

Code

p{ font-size: 17px; } p.tad-larger-font { font-size: calc(17px + var(--tad)); }

a little

make the p tag font-size a litle smaller

This is a litle smaller

Code

p{ font-size: 17px; } p.little-smaller-font { font-size: calc(17px - var(--little)); }

a bit

make the p tag letter-spacing (kerning) a bit wider

This is a bit wider

Code

p{ font-size: 17px; } p.bit-wider { letter-spacing: calc(17px - var(--bit)); }

a slightly

make the p tag font-weight slightly heavier and make the font-size slightly larger

This is a slightly heavier and larger

Code

p{ font-size: 17px; } p.slightly-heavier { font-weight: calc(400 + var(--slightly-heavier)); } p.slightly-larger { font-size: calc(17px + var(--slightly)); }

notch

make the p tag font-size a couple of notches larger

This is a couple of notches larger

Code

p{ font-size: 17px; } p.couple-notches{ font-size: calc(17px + (var(--notch) * var(--couple))); }

code
